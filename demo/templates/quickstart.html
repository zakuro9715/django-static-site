Install django-static-site with pip

pip install django-static-site

Add 'static_site' to INSTALLED_APPS in settings.py

INSTALLED_APPS = (
  ...
  'static_site',
)

Make a 'index.html' file in your template directory.

<!DOCTYPE html>
<h1>Hello World</h1>

Run testing server

./manage.py runserver

Access to http://localhost:8000/
